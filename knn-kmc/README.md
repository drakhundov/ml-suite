# KNN & KMC Classifiers

A lightweight implementation of two classic machine learning classifiers from scratch using NumPy.

## Features

- **KNNClassifier**: k-nearest neighbors classifier with lazy learning. Uses Euclidean distance to classify new samples based on the k closest training examples.
  
- **KMCClassifier**: k-means clustering classifier. Iteratively partitions data into k clusters by minimizing within-cluster variance with early convergence support.

Both implementations use optimized NumPy operations for efficient distance calculations.
